# Nikkie AI CBE Learning Platform

**Nikkie AI CBE** is a modern, interactive, and offline-capable learning platform designed for senior school students following the Competency-Based Curriculum (CBE). It provides notes, quizzes, projects, and interactive dashboards for students, teachers, and administrators.

---

## Features

- **Student Dashboard**
  - Access all subjects with notes, activities, quizzes, and projects.
  - Track learning progress and reflect on key concepts.
  - Competency-based content aligned with the CBE curriculum.

- **Teacher Dashboard**
  - Add lesson notes for different subjects.
  - View student reports and performance analytics.
  - Easily manage content for classes.

- **Master Admin Dashboard**
  - Optimize system performance.
  - Clear all stored data safely.
  - Manage platform settings.

- **Subjects Included**
  - Mathematics
  - Physics
  - Chemistry
  - Biology
  - English
  - ICT
  - Business Studies
  - Geography
  - History & Citizenship
  - Sports Science
  - Visual Arts
  - Performing Arts

- **Additional Features**
  - Dark/Light theme toggle for eye-friendly reading.
  - Matrix-style hacker aesthetic interface.
  - Offline support via service worker (`sw.js`).
  - Fully responsive design for desktop and mobile devices.
