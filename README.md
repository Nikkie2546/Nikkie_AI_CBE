# Nikkie Learning Platform v13 - Offline

**Nikkie Learning Platform** is an offline-capable educational web platform for students and teachers. It runs entirely in the browser, with interactive notes, quizzes, and auto-saving features — no internet required after the first load.  

[**Try it here**](https://nikkie2546.github.io/Nikkie_AI_CBE/)

---

## Features

### For Students
- Login/Signup with name, school, and county.  
- Access notes for core subjects: Mathematics, Physics, English, ICT.  
- Take quizzes and track progress.  
- Personal notes auto-saved in the browser.  

### For Teachers
- Secure login (`admin123`) for managing notes.  
- Add/Edit notes accessible to all students.  
- View student reports (optional).  

### Master Admin (Private)
- Full system control is reserved for internal use.  
- Public users cannot access this feature.  

### Offline Support
- Service Worker caches the platform for offline use.  
- Works without internet after first visit.  

### Settings
- Toggle between dark and light themes.  

---

## Installation / Usage

1. Open [GitHub Pages URL](https://nikkie2546.github.io/Nikkie_AI_CBE/) in a browser.  
2. Use the platform as a student or teacher.  
3. No additional setup needed — works offline automatically.

## Built With
